My Flix app

This web app provides users with access to movie details, directors, and genres. Users can register, update their info, and create a list of favorite movies.

MyFlix API Documentation

Business Logic URL HTTP Method Request body data format Response body data format
Get data about a single movie by title /movies/:title GET None A JSON object holding all info data of a single movie, containing title, year, image url, a brief description, director info and the genre. Example: { "title": "Coach Carter", "year": "2005", "image": "https://www.themoviedb.org/t/p/original/y3HOTTyM5nLsdUzXFtFCohG28qj.jpg", "description": "Coach Carter is a riveting sports drama that follows the inspiring true story of Coach Ken Carter, who shocks the basketball community by benching his undefeated team to prioritize academics, teaching them life lessons that transcend the court and ignite a powerful journey of self-discovery and triumph.", "director": [ { "Name": "Thomas Carter", "Birth": "1953", "Death": "", "Bio": "Thomas Carter is an accomplished American film and television director, known for his work on 'Coach Carter' and other notable projects. With a career spanning decades, he has made significant contributions to the entertainment industry." } ], "genres": [ { "genreName": "Drama", }, ] }, ];
Return data about a genre by name /movies/genre/:genreName GET None A JSON object holding data about a single genre, containing genre name and a brief description. Example: "genres": [ { "genreName": "Drama", "genreDescription": "Drama is a genre of fiction characterized by discourses or works intended to evoke intense emotions, often focusing on the profound and serious aspects of human experience, compelling audiences through compelling narratives and emotional resonance." }, ]
Return data with info about a specific director /movies/directors/:directorsName GET None A JSON object holding data about a single director, containing the director's name, birth year, year of death (if applicable) and a brief bio. Example: "director": [ { "Name": "Thomas Carter", "Birth": "1953", "Death": "", "Bio": "Thomas Carter is an accomplished American film and television director, known for his work on 'Coach Carter' and other notable projects. With a career spanning decades, he has made significant contributions to the entertainment industry." }
Allow new users to register /users POST A JSON object holding data about the user to add, structured as follows: {id: 1, name: "George", email: "george.miller@gmail.com", password: "george123" } A JSON object holding user data plus an ID: favorite movies: { id: 1, name: "George", email: "george.miller@gmail.com", password: "george123", favoriteMovies: ["Coach Carter"] }
Allow users to update their user info (username) /users/:username PUT A JSON object holding the user info that is suspect to change. It should contain at least one of the following key/value pairs: {id: 1, name: "George", email: "george.miller@gmail.com", password: "george123", favoriteMovies: ["Coach Carter"] } A JSON object holding data about the updated user information. Example: { "id": 2, "name": "Lara", "favoriteMovies": [] }
Add a movie to the user's favorite list /users/:Username/movies/:movieID POST None A JSON object holding all user info with all key/values pair updated: { username: "George", password: "george123", 'userEmail':'george.miller@gmail.com', "FavoriteMovies": ["Coach Carter"], }
Remove a movie from the user's favorite list /users/:Username/movies/:movieID DELETE None A JSON object holding all user info with all key/values pair updated: { username: "George", password: "george123", 'userEmail':'george.miller@gmail.com', "FavoriteMovies": [" "], }
Delete user registration /users/:Username DELETE None A text message indicating whether a user was successfully removed.
